# Privacy Policy Site

This folder contains a static privacy policy page ready for GitHub Pages.

## Files

- `index.html`: public privacy policy page

## Before publishing

Replace the placeholder contact email in `index.html`:

- `your-support-email@example.com`

## Fastest GitHub Pages deployment

1. Create a new public GitHub repository, for example `gift-lounge-privacy`.
2. Upload the contents of this folder so that `index.html` is at the repository root.
3. In GitHub, go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save and wait a few minutes.
6. Your public privacy policy URL will look like:
   - `https://<your-github-username>.github.io/gift-lounge-privacy/`

## Notes

- The privacy policy URL must be public and must open directly in a browser.
- Do not upload a PDF instead of this HTML page.
- Keep the page available after your app goes live.
